Human-Assisted Machine Learning

Problem Statement

  • Provide rich reward functions to machine learning algorithms in a CPS that are generated by a human-in-the-loop while minimizing the burden on the human
  • Intrinsic human feedback is captured using EEG based brain waves, inspired by a high-level error-processing system in humans that generates error-related potential/negativity (ErrP)
